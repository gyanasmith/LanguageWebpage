What is UX/UI?

UX means User Experience, and UI refers to User Interface. They both have to do with computers and the person using the computer. Here, you'll explore the different careers in the field of Web Development, User Experience, and all the skills you'll want to know!




UX Design

(User Experience)

User Experience (UX) Designers focus on the overall feel of a product. They make sure that the product is user friendly, makes sense, and provides a good experience when the user is using it.

There are a lot of things that a UX Designer focuses on, including:

  • Empathize: Research and see what users need the most
  • Define: Define what will work best and what won't be important
  • Ideate: Layout of a visual design
  • Prototype: Create a mockup
  • Test: Let people use it and see if its user friendly.

UX Designers should also have a solid set of skills like:
  • Communication skills
  • Organization and time management
  • Presentation skills
Careers that many UX Designers have are:
  1. Front End Developer
  2. Product Designer
  3. UX Engineer
  4. UX researcher
  5. Information Architect




UI

(UI Developer)

UI Designers are responsible for creating the visual layout, aesthetics, and usability of websites. They combine creative skills with their knowledge to build websites that are both functional and nice to look at.

Some of the main things that UI Developers do are:

  • Research
  • Design
  • Code
  • Test
  • Collaborate

They typically create the visual and user interface side of apps and websites.

The careers for UI Developers are pretty much the same as UX Designers, and they include:

  1. Product Designer
  2. Web Developer
  3. Visual Designer
  4. Video Game Creator

and much more.
